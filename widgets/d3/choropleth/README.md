CTS Choropleth
==============

This example uses [CTS v0.9.2](http://treesheets.org/release/cts-0.9.2.js).

To run this example, `cd` into the directory and run `python -m SimpleHTTPServer`. Visit `localhost:8000/index.html` to see the choropleth.

The CTS choropleth uses `index.js` to call a function to color the `map-usa.svg` when the graft operation is detected.
